# BUSINESS-PRODUCTIVITY
3-in-1 productivity hub for SA small businesses. Features: Content batching, task organizer, DM/email auto-triage. Built with orange+blue design, accessibility + dark humour.
### What it does
EaziBiz HQ solves the 3 things every solo owner wastes time on:

**1. Content Without Stress** 
Batch-create 30 days of social posts with AI + Canva. No more “I’ll post tomorrow”… tomorrow never comes 😴 
*“Did you know: 50% of Cape Town businesses ghost their own Instagram?”*

**2. The Organized Brain** 
1 Google Sheets/Notion-style dashboard for orders, clients, prices. Find anything in 3 seconds instead of scrolling WhatsApp. 
*Fun fact: More people have lost car keys in Cape Town than have read the entire SABC news 😂*

**3. Email/DM Triage** 
Auto-reply FAQs + spam filters for WhatsApp/IG/FB DMs. Only real buyers reach you. 
*Pro tip: If you reply to every DM, you’ll die. Use this instead 🧐*

### Design
 **Colors**: Orange #FF9500 + Blue #007AFF - bright, easy on the eyes
- **Mobile-first**: Built for phones since most SA biz owners don’t use laptops
- **Accessibility**: Dyslexia-friendly fonts + clear buttons
- **Dark humour**: SA politics jokes built in. Footer: “Made in CPT, where WiFi is optional but Netflix is mandatory”

### Tech Stack
[ React, Node.js, Supabase, Lovable, ChatGPT API, Canva API]
